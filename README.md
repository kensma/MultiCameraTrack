# 多相機追蹤
## 安裝
### 環境
- Python 3.8
### 套件安裝
```bash
pip install -r requirements.txt
```
### PyTorch安裝
```bash
conda install pytorch==1.13.1 torchvision==0.14.1 torchaudio pytorch-cuda=11.7 -c pytorch -c nvidia
```
### cython_bbox安裝
```bash
pip install cython_bbox
```
